## Highlights

### ☸️ Kubernetes Self-Healing (Mini Project 07)
An interactive calculator application deployed to Minikube running `3` replicas with a helper PowerShell script to randomly delete active pods and demonstrate Kubernetes automatically healing and restoring the desired state in under 2 seconds.

### 📈 Horizontal Pod Autoscaling (Mini Project 08)
Deploys a CPU-intensive calculator web application and configures an HPA to dynamically scale from `2` to `8` replicas based on average CPU target load, stress-tested using the `hey` load testing tool.
